## 实践流程
1. 下载安装Git
2. 配置用户名和邮箱
3. git init 创建本地仓库
4. 在GitHub创建远程仓库
5. git remote 关联远程仓库
6. 完成4次提交
7. git push 推送到远程

## 四次提交说明
1. 第一次：创建README.md文件
2. 第二次：添加test.py测试代码
3. 第三次：完善实践笔记内容
4. 第四次：添加本人自主研发的游戏代码game.py

## 遇到的问题及解决方法
1. 直接输入仓库链接报错：必须配合git remote add origin使用
2. 提交失败：需要先git add . 再执行commit
